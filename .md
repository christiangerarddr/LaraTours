## Features

-   Laravel 9
-   Vue + VueRouter + Vuex
-   Login, register, update profile
-   password reset
-   Authentication with Sanctum
-   Sample Crud
-   Tailwind + Heroicons

## Installation

-   `cp .env.example .env`
-   on your `.env`, set your database connection details and app_url
-   `composer install`
-   `php artisan migrate`
-   `php artisan key:gen`
-   `npm install`
-   `npm run dev`

#### Development

```bash
npm run watch

```
